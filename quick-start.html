Quick Start

  • Open a file temporarily from the Scores menu, or add your own score files to the library.
  • Best compatibility comes from MusicXML / MXL.
  • You can also import MIDI, MuseScore 3, and Guitar Pro 5 files. These are converted in-browser before loading.
  • Connect your MIDI keyboard in Settings.
  • On iPad / iPhone, use midiweb.cc for MIDI input support.
  • LED output is optional. WLED can use HTTP JSON, or DDP with the local helper for lower latency.
  • Starter songs are added on first run and can be deleted.

Library & Backups

  • Your score library is stored in your browser.
  • Use Scores → Backup Library to back up your song library.
  • Use Settings → Backup All Settings to save trainer preferences, connections, and LED calibration.

Score Files

  • Recommended formats: .xml, .musicxml, and .mxl.
  • Also supported through built-in conversion: .mid, .midi, .mscz, .mscx, .gp, .gp3, .gp4, .gp5, .gpx, .gtp, and .ptb.
  • If a converted file loads imperfectly, try exporting it as MusicXML first.
  • Browse free starter songs at musetrainer.github.io/library.

Important Note

Some browsers may clear local site storage, especially on mobile or low-storage devices. Back up your library and LED calibration if the data matters to you.
