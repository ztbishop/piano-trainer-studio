Quick Start

  • Open a score from the Scores menu, or add your own files to the library.
  • Best results: MusicXML / MXL.
  • Also supported: MIDI, MuseScore, and Guitar Pro files. These are experimental, and MIDI does not include all notation details.
  • Connect your MIDI keyboard in Settings.
  • On iPad / iPhone, use midiweb.cc for MIDI input support.
  • LED output is optional. See the setup guide for hardware and setup details.
  • Starter songs are added on first run and can be deleted.
  • Browse free starter songs at musetrainer.github.io/library.

Library & Backups

  • Your score library is stored in your browser. Back it up if it matters to you.
  • Use Scores → Backup Library to back up your song library.
  • Use Settings → Backup All Settings to save trainer preferences, connections, and LED calibration.

Important Note

Some browsers, especially on mobile or low-storage devices, may clear local site data over time. Back up your library and LED calibration if the data matters to you.
