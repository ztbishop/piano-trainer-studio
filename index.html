Help / About

Quick Start

  • Open a score from the Scores menu, or import your own MusicXML files.
  • Connect your MIDI keyboard in Settings.
  • On iPad / iPhone, use midiweb.cc for MIDI input support.
  • LED output is optional. WLED can use HTTP JSON, or DDP with the local helper for lower latency.
  • Starter songs can be included on first run when the built-in starter library is available.

Library & Backups

  • Your score library is stored in your browser.
  • Use Scores → Export Library to back up your song library.
  • Use Settings → Backup All Settings to save trainer preferences, connections, and LED calibration.

MusicXML Sources

  • Browse free starter songs at musetrainer.github.io/library.
  • Use tools like MuseScore to convert MIDI to MusicXML or transpose songs before importing.

More Help

For deeper help, LED strip setup notes, and WLED guidance, open the Full Setup Guide.

Important Note

Some browsers may clear local site storage, especially on mobile or low-storage devices. Back up your library and LED calibration if the data matters to you.
